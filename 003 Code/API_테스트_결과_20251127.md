## 핵심 기능 테스트 상세

### 1. 추천 API 결과 (GET /api/v1/users/me/recommendations)

```json
{
  "recommendedMenus": [
    {
      "menuId": 250,
      "menuName": "다래식혜",
      "restaurantName": "다래촌",
      "similarity": 0.98
    },
    {
      "menuId": 287,
      "menuName": "진순자만두국",
      "restaurantName": "진순자만두",
      "similarity": 0.98
    },
    {
      "menuId": 60,
      "menuName": "치즈 갈비떡볶이",
      "restaurantName": "다해어죽",
      "similarity": 0.98
    }
  ]
}
```

> **핵심**: 98% 유사도로 추천 동작! 시연시 "내 취향 98% 일치" 멘트 가능

---

### 2. 주문 생성 테스트 (POST /api/v1/orders)

**요청**:
```json
{
  "orderType": "DELIVERY",
  "orderMenus": [
    { "menuId": 56, "quantity": 1 }
  ]
}
```

**응답** (200 OK):
```json
{
  "orderId": 2,
  "totalPrice": 10000,
  "status": "PENDING",
  "type": "DELIVERY",
  "createdAt": "2025-11-27T12:34:39.702501",
  "orderMenus": [
    {
      "menuId": 56,
      "menuName": "다해어죽",
      "menuPrice": 10000,
      "quantity": 1
    }
  ]
}
```

## 백엔드 팀 추가 작업 필요

### ⚠️ 필수: 쿠폰 데이터 추가

현재 쿠폰이 0개입니다. 시연을 위해 아래 쿠폰 추가 필요:

**Admin API**: `POST /api/v1/admin/coupons`
```json
{
  "name": "신규 가입 쿠폰",
  "code": "WELCOME3000",
  "discountAmount": 3000,
  "totalQuantity": 100,
  "restaurantId": null
}
```

### ⚠️ 선택: 공지사항 추가

시연 시 공지사항을 보여줄 경우:

**Admin API**: `POST /api/v1/admin/notices`
```json
{
  "title": "MobiSync 오픈 기념 이벤트",
  "content": "맛 취향 설정하고 3000원 쿠폰 받아가세요!"
}
```

## 시연 가능성 결론

### ✅ 시연 가능한 기능
1. 로그인/회원가입
2. 맛 취향 설정 (슬라이더)
3. 추천 메뉴 확인 (98% 유사도!)
4. 레스토랑/메뉴 조회
5. 찜 목록 관리
6. 주문 생성/조회

### ⚠️ 쿠폰 기능
- 현재 쿠폰 데이터가 없어서 쿠폰 적용 시연 불가
- **해결책**: 백엔드 팀이 Admin API로 쿠폰 추가 또는 시연에서 쿠폰 부분 생략

### 🎯 권장 시연 흐름
1. 로그인 (demo2@test.com)
2. 맛 설정 슬라이더 조절
3. 홈에서 "98% 일치" 추천 결과 보여주기 ⭐
4. 다해어죽(restaurant_id: 23) 상세 페이지
5. 메뉴 담기 → 주문 생성
6. 찜 목록 확인
7. 주문 내역 확인
